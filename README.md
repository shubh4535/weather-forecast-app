# Weather Forecast Application

This is a simple and responsive Weather Forecast web application built using
HTML, Tailwind CSS, and JavaScript. The application fetches real-time weather
data from the OpenWeatherMap API and displays current weather conditions along
with an extended forecast.

## Features

- Search weather by city name
- Get weather for current location using Geolocation API
- Display temperature, humidity, and wind speed
- Toggle temperature between Celsius and Fahrenheit
- 5-day weather forecast
- Weather condition icons and dynamic background changes
- Recent searched cities dropdown using localStorage
- Graceful error handling with UI messages

## Tech Stack

- HTML
- Tailwind CSS
- JavaScript
- OpenWeatherMap API

## How to Run the Project

1. Clone the repository
2. Open `index.html` using Live Server (recommended)
3. Enter a city name or use current location

## Notes

- LocalStorage is used to store recently searched cities
- No external frameworks were used
- Node modules are not required
